# Spring MVC Project

## Overview

This is a simple Spring MVC application built to understand the fundamentals of Spring MVC architecture.

## Features

- Add Employee
- View Employee Details
- Update Employee Information
- Delete Employee Records
- MVC Architecture
- JSP Frontend
- MySQL Database Integration

## Technologies Used

- Java
- Spring MVC
- Hibernate/JPA
- JSP
- HTML
- CSS
- MySQL
- Maven
- Eclipse IDE

## Project Structure

- Controller Layer
- Service Layer
- DAO Layer
- Model Layer
- View Layer (JSP)
